type of instructions
1 Arithmetic instructions (+ - / *)
2 data type declaration instructions
3 control instructions (if..else)
4 looping instructions (for...while...do while)
5 I/O instructions

VS Code is IDE (Integrated Development Envirnment)

steps for creating JS project
create blank folder and open CMD
project creation===npm init -y
open folder in VS code using code .
package installation == npm install nodemon
create demo.js file
add start script in script block of package.json file
"start":"nodemon demo.js"
run program ==npm start

JS Operators
Arithmetic Operator --: + - / * %
Assignment Operator --: = += -= /= *= %=
Logical Operator --: &&(and) ]](or) !(not)
Comparison/relational Operator --: >(lt) <(gt) >=(gte) <=(lte) ==(equal to) !=(not equal)
